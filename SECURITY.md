# Security Policy

SCI is a sandboxed interpreter. If you use it to evaluate untrusted code, security
issues (sandbox escapes) are taken seriously.

## Reporting a vulnerability

Please report vulnerabilities privately via GitHub's
[private vulnerability reporting](https://github.com/babashka/sci/security/advisories/new)
rather than opening a public issue. You can expect an initial response within a few days.

## Supported versions

Security fixes are applied to the latest released version. Please upgrade to the
latest release to receive them.
